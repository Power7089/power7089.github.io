Frida在Windows下使用tar包方式安装过程记录

那是一个懒惰的下午，接到测试任务，要对某APP展开测试，万事具备，只差一个Frida==15.1.17版本。

万万没想到，踩了两个小时的坑。

看似简单，实则困难重重……

Frida 常规的安装方法是pip3 install frida==15.1.17(指定版本)，但这各种方法各种报错，排查原因折磨了我半天，也许是网络原因，也许是版本原因，绕来绕去太麻烦，脱离了只是为了安装指定版本frida的目的。

试了一大圈，找到一种安装方式，大家可以试试。

①、从pypi处下载frida15.1.17的tar包。https://pypi.org/project/frida/15.1.17/#files

②、下载的过程，进入cmd安装依赖环节。先安装pip3 install wheel，不确定是否有影响，但安装总没错。

③、pip3 install frida-tools，安装firda-tools会自动下载最新版本frida。安装成功我们需要卸载。

④、卸载最新frida pip3 uninstall frida

⑤、解压刚才下载的tar包。cmd进入该目录，输入命令py -3 setup.py bdist_egg

⑦、cmd 输入 frida –version 查看是否安装完成。
